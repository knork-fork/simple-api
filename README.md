# Simple API

Lightweight CRUD wrapper for PostgreSQL database.

### Installation

Start container:

```
docker-compose up --build -d
```

Run composer install:

```
docker/composer install --no-interaction
```


By default app should be available @ `localhost:11000`

Or if already built:

```
docker-compose up -d
```

Stop container:

```
docker-compose down
```
